# Blog

blog在线预览地址：http://139.224.74.133

前端是react+redux+react-router
后端是node+express+mongoDB提供的API

### 开始 
1.安装 nodejs(>=4) 
# 2.启动mongodb

### 运行
    开发环境===>
    npm run node
    npm run dev
    (上面2个命令开2个窗口同时运行)
    浏览器输入 localhost:7070

    生产环境===>
    npm run node
    npm run dist
    浏览器输入 localhost:8080
